


# Setup

> Starting a new application? Our boilerplate, [Liberty](https://bitbucket.org/lendi-dev/liberty/src/master/) has `lendi-ui` set up out-of-the-box.

## Installation

Install `@lendi-ui/reset`, `@lendi-ui/theme` and whichever other packages you wish to use e.g. `@lendi-ui/logo`.

```
yarn add @lendi-ui/reset @lendi-ui/theme @lendi-ui/logo
```

## Usage

1. Import `@lendi-ui/reset` and `@lendi-ui/theme` into your root file:

  ```
  import '@lendi-ui/reset';
  import Theme from '@lendi-ui/theme';
  ```


2. Wrap your root component with the theme:

  ```
  ReactDOM.render(
    <Theme>
      <App/>
    </Theme>,
    document.getElementById('app')
  );
  ```

3. Import and use one of the other packages:

  ```
  import Logo from '@lendi-ui/logo';

  <Logo variant="dark"/>
  ```
