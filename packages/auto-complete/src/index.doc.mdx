## Usage

```
import AutoComplete from '@lendi-ui/auto-complete';
```
// Please see the working examples. There are some positioning issues with dropdown on this playground page. 
// Search keyword like - 'Bank' 
```jsx
<>
 
With Promise
<AutoComplete dataSource={(userInput) => {
return new Promise(function(res, rej){
            const filteredDataSource = [{
    label: 'AMP Bank',
    value: 'AMP',
  },
  {
    label: 'ANZ Bank2',
    value: 'anz',
  }].filter((d) => d.label.toLowerCase().indexOf(userInput.toLowerCase()) > -1);
                return res(filteredDataSource);
    })
}} isFullWidth={true}/>
Search keyword like - 'Bank'
<br/>
Without Promise
<AutoComplete dataSource={[{
    label: 'AMP Bank',
    value: 'AMP',
  },
  {
    label: 'ANZ Bank2',
    value: 'anz',
  }]} isFullWidth={true}/>
<br/><br/>
</>

```
