# @lendi-ui/footer

Lendi website footer component.

## Installation

```
yarn add @lendi-ui/footer
```

```jsx
<Footer />

## Properties
<PropTable>
  <PropTable.Entry name="broker" type="object" description="Assigned broker data"/>
  <PropTable.Entry name="params" type="string" description="Accept query parameters from users. This one will be deprecated soon."/>
</PropTable>

## Notes
The Footer component can also support query parameters currently. But it will be deprecated soon.
<b>Support for native props like Aria attributes, Standard HTML Attributes like title, classname, id, role, itemProp, itemID, itemRef</b>
```
