## Usage

```
import { Card } from '@lendi-ui/card';
```

```jsx
<Card>
  <Card.Header>Ex fastidii perseque</Card.Header>
  <Card.Content>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. In cursus lacus ut urna entum sodales.
    In cursus lacus ut urna entum sodales. In cursus lacus ut urna entum sodales.
  </Card.Content>
  <Card.Footer align="right">
    <ModeEdit color="primary.500" width="1.5em" height="1.5em" />
    <Delete color="primary.500" width="1.5em" height="1.5em" />
  </Card.Footer>
</Card>
```

## RHS button content

Optimal order of appearance: Negative, Neutral, Positive (left to right)

  Negative actions:

  ```
  No, Cancel, Delete, Close, Hide, Dismiss, Reset, Back, Previous
  ```
  Neutral actions:

  ```
  Not sure, Edit, Save, Help, Export, Upload, Download, Move, Settings, Options, Browse, 
  Preferences, Copy, Preview
  ```
  Positive actions:

  ```
  Yes, Next, Continue, Accept, OK, Commit, Submit, Complete, View, Show, Add, Start, Create, 
  Contact, Learn more, Confirm, Open, Send, Enable, New
  ```
