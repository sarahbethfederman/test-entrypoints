# link

Styles for links.

## Mixin

```
import {link} from '@lendi-ui/typography';
```

```js
const A = styled.a`
  ${link()}
`;

render(<>The <A>quick brown fox</A> jumps over the <A>lazy dog</A>.</>)
```
