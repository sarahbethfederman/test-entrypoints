# overline

Styles for overline copy.

## Mixin

```
import {overline} from '@lendi-ui/typography';
```

```js
const Text = styled.div`
  ${overline({size: 'lg', colorScheme: 'light'})}
`;

render(<Text>Default overline</Text>)
```

### Options

<PropTable>
  <PropTable.Entry name="size" type={'"sm" | "md" | "lg" | "xl"'} required description="The overline copy size."/>
  <PropTable.Entry name="colorScheme" type={'"light" | "dark"'} defaultValue="light" description="Determines the colour scheme for the overline copy."/>
  <PropTable.Entry name="align" type={'"left" | "center" | right"'} defaultValue="left" description="Horizontal alignment of overline copy."/>
</PropTable>


## Component

```
import {Overline} from '@lendi-ui/typography';
```

```jsx
<>
  <div style={{padding: '1rem'}}>
    <Overline size="md" colorScheme="light">Default Overline</Overline>
  </div>
  <div style={{backgroundColor: 'darkslategrey', padding: '1rem'}}>
    <Overline size="md" href="#" colorScheme="dark">Dark Overline with href</Overline>
  </div>
</>
```
