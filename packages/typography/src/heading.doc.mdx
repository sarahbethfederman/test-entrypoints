# heading

Styles for heading copy.

## Mixin

```
import {heading} from '@lendi-ui/typography';
```

```js
const Text = styled.div`
  ${heading({size: 'lg', color: 'warn.500'})}
`;

render(<Text>The quick brown fox jumps over the lazy dog</Text>)
```

### Options

<PropTable>
  <PropTable.Entry name="size" type={'"xl" | lg" | "md" | "sm" | "xs"'} required description="The heading copy size"/>
  <PropTable.Entry name="color" type={'string'} description="The heading copy color"/>
</PropTable>

## Component

```
import {Heading} from '@lendi-ui/typography';
```

```jsx
<Heading size="lg" color="warn.500">The quick brown fox jumps over the lazy dog</Heading>
```
