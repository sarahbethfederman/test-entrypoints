## Usage

```
import Modal from '@lendi-ui/modal';
```

```jsx
// Note: setting the prop "show" to "true" will not open the modal here. For a working example you can go to the example page, linked above.

  <Modal isVisible={false} size="md" onHide={this.onHide} id="modalId">
    <Modal.Content>
      <Body>
        You already started an application on the 15th of May, 2019.
      </Body>
      <br />
      <Body>
        Would you like to continue?
      </Body>
    </Modal.Content>
    <Modal.Footer>
      <ButtonGroup isFullWidth size="sm">
        <ButtonGroup.Button variant="secondary">Continue application</ButtonGroup.Button>
        <ButtonGroup.Button variant="primary">Start a new application</ButtonGroup.Button>
      </ButtonGroup>
    </Modal.Footer>
  </Modal>
```
