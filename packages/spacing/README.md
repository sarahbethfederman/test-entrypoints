# @lendi-ui/spacing

Spacing styles HTML element spacing like margin and padding.

## Installation

```
yarn add @lendi-ui/spacing
```

## Usage

- margin

```jsx
import { m } from '@lendi-ui/spacing';
const Box = styled.div`
  ${m('lg')};
`;
```

- padding

```jsx
import { p } from '@lendi-ui/spacing';
const Box = styled.div`
  ${p('lg')};
`;
```
