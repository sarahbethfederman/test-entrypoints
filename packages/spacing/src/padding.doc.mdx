# Padding

Utilities for applying margin

## Mixin
```
import {p, px, py, pl, pb, pr, pt} from '@lendi-ui/spacing';
```

## Usage
```js
const Wrapper = styled.div`
  background-color: tomato;
  ${p("lg")}
`;

const Box = styled.div`
  background-color: #fff;
`;

render(
  <Wrapper>
    <Box>The quick brown fox jumps over the lazy dog</Box>
  </Wrapper>
)
```

Can also be used with breakpoints:
```js
const Wrapper = styled.div`
  background-color: tomato;
  ${p({ "mobile": "xxs", "tablet": "md", "desktop": "xxl" })}
`;

const Box = styled.div`
  background-color: #fff;
`;
```
