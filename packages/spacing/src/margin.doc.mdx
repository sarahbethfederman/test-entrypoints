# Margin

Utilities for applying margin

## Mixin 

```
import {m, mx, my, ml, mb, mr, mt} from '@lendi-ui/spacing';
```

## Usage
```js
const Wrapper = styled.div`
  border: 1px dotted tomato;
`;

const Box = styled.div`
  ${m("lg")}
`;

render(
  <Wrapper>
    <Box>The quick brown fox jumps over the lazy dog</Box>
  </Wrapper>
)
```

Can also be used with breakpoints:
```js
const Wrapper = styled.div`
  border: 1px dotted tomato;
`;

const Box = styled.div`
  background-color: #fff;
  ${m({ "mobile": "xxs", "tablet": "md", "desktop": "xxl" })}
`;

)
```
