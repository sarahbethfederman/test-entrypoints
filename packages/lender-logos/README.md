# @lendi-ui/lender-logos

A LenderLogos component.

## Installation

```
yarn add @lendi-ui/lender-logo
```

## Usage

```jsx
import LenderLogos from '@lendi-ui/lender-logos';

<LenderLogos lenderId="" />;
```

or

```jsx
import BankAustralia from '@lendi-ui/lender-logos/BankAustralia';

<BankAustralia />;
```
