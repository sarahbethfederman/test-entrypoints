## Usage

```
import Popover from '@lendi-ui/popover';
```

```jsx
<>
  <div>
    <Popover
      content={<><Body size='sm' mb='xxs'>Heading label</Body><Body size='xs'>This is LUI popover conten body. You can input extra infomation here.</Body></>}
      position='top'
      target={<Body variant="primary" size="sm" style={{display: 'inline'}}><Question color="shade.1000" /> You can trigger a popover on any React Component</Body>}
    />
  </div>
  <Popover
    content={<Body size='sm' mb='xxs'>I pop out from the left</Body>}
    position='left'
    target={<ChevronLeft color="secondary.600" />}
  />
</>
             
```
### Design Notes
Popovers are meant to give related additional content. They can be much more verbose (compared to Tooltips) and they can include a header and many lines of text in the body.
