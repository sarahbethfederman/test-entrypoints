# unit

A grid unit.

## Mixin

```
import { unit } from '@lendi-ui/grid';
```

```js

const Grid = styled.div`
  ${grid()}
`;

const Quarter = styled.div`
  ${unit({offset: 1/4, size: 1/4})}
`;

const Eighth = styled.div`
  ${unit({size: {xs: 1/2, sm: 1/4, lg: 1/8}})}
`;

const Content = styled.div`
  text-align: center;
`;

const QuarterContent = styled(Content)`
  background: tomato;
`;

const EighthContent = styled(Content)`
  background: peachpuff;
`;

render(
  <Grid>
    <Eighth><EighthContent> ⅛ </EighthContent></Eighth>
    <Quarter><QuarterContent> ¼ </QuarterContent></Quarter>
    <Eighth><EighthContent> ⅛ </EighthContent></Eighth>
  </Grid>
)
```

### Options

Note all props can consume (BreakpointValueMap)[/packages/breakpoint] which allows you to specify values per breakpoint.

<PropTable>
  <PropTable.Entry name="offset" type="number | BreakpointValueMap<number>" defaultValue={0} description="The size of the grid unit offset as a fraction"/>
  <PropTable.Entry name="size" type="number | BreakpointValueMap<number>" defaultValue={1} description="The size of the grid unit as a fraction"/>
</PropTable>
