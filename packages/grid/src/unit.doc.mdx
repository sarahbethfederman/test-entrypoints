# unit

A grid unit.

## Mixin

```
import { unit } from '@lendi-ui/grid';
```

```js

const Grid = styled.div`
  ${grid()}
`;

const Quarter = styled.div`
  ${unit({offset: 1/4, size: 1/4})}
  text-align: center;
  background: tomato;
`;

const Eighth = styled.div`
  ${unit({size: 1/8})}
  text-align: center;
  background: peachpuff;
`;

render(
  <Grid>
    <Eighth> ⅛ </Eighth>
    <Quarter> ¼ </Quarter>
    <Eighth> ⅛ </Eighth>
  </Grid>
)
```

### Options
