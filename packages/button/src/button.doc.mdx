## Usage

```
import {Button} from '@lendi-ui/button';
```

```jsx
<>
<Button variant="primary" size='xs' id="buttonId" aria-label="buttonLabel">Continue</Button>
<br/><br/>
<Button variant="primary">Continue</Button>
<br/><br/>
<Button variant="secondary" before={<SentimentVerySatisfied color='primary.500' width='1.5em' height='1.5em' />} after={<Close color='primary.500' width='1.5em' height='1.5em' />}>Continue</Button>
<br/><br/>
<Button variant="primary" ariaLabel="locked"><Lock color="shade.0"/></Button>
<br/><br/>
<Button variant="primary" size={{mobile:'sm', tablet:'md', desktop:'lg'}}>Continue</Button>
</>
```

## Variants 
### Primary
For important, mandatory or positive actions. 
```jsx
<Button variant="primary" size='sm'>Continue</Button>
```

### Secondary
For actions that are not mandatory or as important or as positive as the primary funtion. 
May also be used in conjuction with a primary button to help a user make the best choice.
```jsx
<Button variant="secondary" size='sm'>Continue</Button>
```

### Emphasis
Used in rare circumstances where an action is critical to the success of the users goals.
```jsx
<Button variant="emphasis" size='sm'>Continue</Button>
```

### Empty
Used when adding a UI element. Most often used with an icon, but not exclusively.
```jsx
<Button variant="empty" size='sm'>Continue</Button>
```
