## Usage

```
import {Radio} from '@lendi-ui/radio';
```

```jsx

<Radio value='1' label='LUI single Radio Button' onChange={() => {}} />

```

### Notes 
Radio should only be used when there are two or more options. If there is a single option, consider using a checkbox instead.
