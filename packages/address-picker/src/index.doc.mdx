## Usage

```
import AddressPicker from '@lendi-ui/address-picker';
```

```jsx

<AddressPicker onSelectAddress={alert} onChange={alert} mapType="static" />
             
```

## Notes

When using this component, googlemap api script link is needed. Please ask <strong>HUB team</strong> for this script link.
