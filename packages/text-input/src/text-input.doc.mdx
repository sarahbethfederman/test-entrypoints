## Usage

```
import Input from '@lendi-ui/text-input';
```

```js
const EmailIcon = styled(Email)`
   ${pl('sm')}
  ${pr('xxxs')}
 `;
const AfterIcon = styled(Lock)`
  ${pr('sm')}
  ${pl('xxxs')}
`;

render(
<>
  'xs' size Input
  <br/>
   <Input
    size='xs'
      placeholder="Input here ..."
      onFocus={() =>{}}
      isFullWidth={true}
      before={
        <Search
          color='primary.500'
          width='12px'
          height='12px'
          style={{ marginLeft : '10px' }}
        />}
    />
  <br/><br/>
  <Input
    size='md'
    placeholder="Input here ..."
    onChange={() =>{}}
    isFullWidth={true}
    before={
      <EmailIcon
        color='primary.500'
        height='26px'
      />}
    after={
      <AfterIcon
        color='primary.500'
        height='26px'
      />}
    />
  <br/><br/>
  Input with native props support like disabled
  <br/> 
   <Input
    placeholder="Input here ..."
    onFocus={() =>{alert('focussed')}}
    isFullWidth={true}
    disabled
  />
  <br/>

</>)
```

## Notes
When an icon is used as the before/after item with an input there are recommended padding settings to ensure that it is displayed as initially intended. To apply the appropriate amount of padding, use `pl()` for before icons and `pr()` for after icons as per the below examples:

```
import { styled } from 'styled-components';
import { pl /* or pr for after icons */ } from 'spacing';

// lg sized input e.g.
const Wrapper = styled(IconName)`
  ${pl('md')}
`;

// md sized input e.g.
const Wrapper = styled(IconName)`
  ${pl('sm')}
`;

// sm sized input e.g.
const Wrapper = styled(IconName)`
  ${pl('xs')}
`;

***Text-Input also support default input props like autoFocus, disabled, onFocus, onChange, name and all AriaAttributes.
```
