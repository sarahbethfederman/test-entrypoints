## Usage

```
import Dropdown from '@lendi-ui/inline-dropdown';
```

```jsx

items= [
    {
      value: '1',
      label: '5 years'
    },
    {
      value: '2',
      label: '10 years'
    },
    {
      value: '3',
      label: '15 years'
    },
    {
      value: '4',
      label: '20 years'
    },
    {
      value: '5',
      label: '25 years'
    }
],

 <InlineDropdown
    size={{mobile:'sm', tablet:'md', desktop:'lg'}}
    items={items}
/>



```
